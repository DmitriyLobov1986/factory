Настройки параметров


В форме содержатся флаги (константы) с помощью которых подключаются отдельные возможности конфигурации.

Общие настройки

Использовать группы пользователей

Установка флага позволяет использовать группы в справочниках Пользователи и Внешние пользователи. Использовать группы в указанных справочниках можно, например, для группировки существенного количества пользователей.

Параметры доступа в Интернет на сервере 1С:Предприятия

Команда открывает форму для ввода параметров прокси-сервера для сервера 1С:Предприятия (в случае использовании файлового варианта, такие настройки производятся для каждого рабочего места, раздел Администрирование, команда Персональные настройки).
